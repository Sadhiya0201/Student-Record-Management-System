# Student Record Management System

## Description
This is a Java-based console application designed to manage student records efficiently.  
It uses Object-Oriented Programming principles to organize the data and operations.  
The program supports adding, viewing, searching, updating, and deleting student information, all stored in memory using ArrayList.

## Features
- Add a new student with ID, Name, Department, and Marks
- View all student records
- Search for a student by ID
- Update existing student information
- Delete a student record
- Menu-driven interface for easy navigation

## Technology
- Java (OOP concepts)
- ArrayList for in-memory data storage
